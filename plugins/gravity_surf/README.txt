== Description ==

Gravity Surf enables a personalized surfing experience on any website. Please follow the instructions outlined below to install and activate your plugin.

For additional features or support, please email us anytime at sayhello@gravity.com

== Installation ==

1. Save the plugin to your hard drive.
2. Log in to your WordPress admin panel. Click 'Plugins', then 'Add New', and finally 'Upload'.
3. Click 'Choose File', and browse to and select the plugin zip file.
4. Click 'Install Now' and, once the installation completes, click 'Activate Plugin'.

You did it. Nice work.

NOTE: If you have a page/content caching plugin in use, empty the cache to have plugin start working immediately.
